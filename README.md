# DJ2LS RFChat

Chat application with a focus on "keep it simple" for ARDOP

## Install nodejs 


## Clone Beta Branch
`git clone -b beta https://gitlab.com/DJ2LS/dj2ls-rfchat.git`


## Update existing Branch
`git pull`

## Install dependencies
`npm install`

## Run 
`node start.js`
Access website by localhost:3000
Configure your station by pressing "Settings"
